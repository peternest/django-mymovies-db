MyMovies DB

Ультимативный гайд по миру кино. Навигация

  • /movies/top100/ - 100 лучших фильмов
  • /movies/series-top50/ - 50 лучших сериалов
  • /movies/1/ - Карточка фильма/сериала
  • /movies/add/ - Добавить новый фильм/сериал
  • /admin/ - Админка Django

  • /api/ - Корень API
  • /api/movies/ - Список фильмов (json)
  • /api/countries/ - Список стран (json)
  • /api/directors/ - Список режиссеров (json)
  • /api/genres/ - Список жанров (json)
